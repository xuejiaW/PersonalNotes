# C# Version

不同的 Unity 版本所支持的 C# 版本不同，如 `2020.3` 版本[^1]支持 `C# 8.0`，`2022.1` 版本[^2]支持 `C# 9.0` 等。

Unity 对于支持的 C# 版本，可能存在部分特性的不支持。其不支持的特性版本在各说明页面中都会有相应说明，如下为 `2022.1` 中不支持的 `C# 9.0` 特性[^2]：
