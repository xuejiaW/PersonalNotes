# C# Version

不同的 Unity 版本所支持的 C# 版本不同，如 `2020.3` 版本[^1]支持 `C# 8.0`，`2022.1` 版本[^2]支持 `C# 9.0` 等。

```ad-tip
Unity 对于支持的 C# 版本
```
