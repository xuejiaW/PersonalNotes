`转义序列（Escape Sequences）` 用来在字符串中表示特殊字符，如 `"` 被用来表示字符串，则当要输入 `"` 时，需要使用 `\"`。

转义序列都使用 `\` 作为前缀，且包含如下选项：

| 转义序列 | 字符名称 |
| -------- | -------- |
| `\'`     | 单引号   |
| ``         |   双引号       |
