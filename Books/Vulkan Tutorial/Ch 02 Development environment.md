# Visual Studio

## Vulkan SDK

从 [Lunary 官网]([LunarXchange (lunarg.com)](https://vulkan.lunarg.com/)) 下载并安装 Vulkan SDK：
![](assets/Ch%2002%20Development%20environment/image-20220224092313285.png)

通常而言，安装后的路径为 `C:\VulkanSDK\<version>`，从中找到 `Bin/vkcube.exe` 程序并运行，如果运行成功，则说明该设备显卡的驱动支持 Vulkan。
